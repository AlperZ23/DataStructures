2.12 The algorithm follows the following steps to sort the list of data: 2 4 1 3
     - Check if the first value is larger then the second if so swap ( new data: 2 4 1 3 )
     - Check if the 2nd value is larger then the 3rd, it is so it is swapped ( new data: 2 1 4 3 )
     - Check if the 3rd value is larger then the 4th, it is so swap ( new data: 2 1 3 4 )
     - Go back to the start because all values have been changed
     - Check if the 1st value is larger than the second, it is so swap ( new data: 1 2 3 4 )
     - The data is now sorted
